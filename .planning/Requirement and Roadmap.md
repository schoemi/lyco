Ich möchte eine Webanwendung erstellen, die mir mit unterschiedlichen Methoden hilft einen Songtext zu erlernen.


# Anforderungen
- Als Nutzer möchte ich unterschiedliche Sets mit mehreren Songs erstellen
- Als Nutzer möchte ich einem Song einen Text hinzufügen
- Als Nutzer möchte ich einen Songtext aus einer PDF importieren
- Als Nutzer möchte ich einen Songtext aus einem Tabulatur-Format für Gitarren importieren
- Als Nutzer möchte ich Songtexte mit unterschiedlichen Methoden erlernen
- Als Nutzer möchte ich einen Punktwert bekommen zu welchem Grad ich einen Songtext auswending kann
- Als Nutzer möchte ich unterschiedliche "Quiz-Interaktionen" nutzen um meine Kenntnisse zu einem Songtext zu überprüfen
- Als Nutzer möchte ich die Anwendung in einem Webbrowser auf unterschiedlichen Geräten (Desktop, ipad, smartphone) nutzen
- Als Nutzer möchte ich MP3-Dateien oder Steaming-Anbieter-Links hinzufügen um die Musik zum Songtext abspielen zu können.
- Das System soll die unterschiedlichen Bausteine eines Songtextes (Chorus, Verse, Pre-Chorus, etc.) verwalten können.
- Das System soll Markups für Pause, Wiederholungen unterstützen
- Das System soll Markups für Gesangstechnik (Atmung, Kopfstimme, Bruststimme, etc.) verwalten und anzeigen können.
- Als Nutzer möchte ich den Songtext und das Markup editieren können.
- Das System soll Markups für Timecodes (z.B. je Baustein) speichern und vewalten können, damit beim der Player für die Audio-Datei bzw. den Webplayer des Streaming-Anbieters an die korrekte Stelle springen kann.
- Als Nutzer möchte ich mich am System anmelden können.
- Als Admin möchte ich User erstellen, bearbeiten und löschen können.
- Das System soll Sets, Songs, Texte in einer Datenbank verwalten
